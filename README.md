# Nivestha

Nivestha is an AI-powered fractional real estate investment platform designed to make property investment accessible and data-driven. The platform allows users to invest in real estate properties through fractional ownership while leveraging machine learning models to predict property prices and rental returns.

The system integrates a full-stack architecture consisting of a React-based frontend, a Node.js backend API, PostgreSQL database management, and a Python-based AI service for property price prediction.

Key Features:

* Fractional property investment system
* AI-based property price prediction
* Interactive dashboard for investors
* Secure backend APIs for transactions
* Property listing and management
* Rental income distribution tracking
* Data visualization and analytics

Technologies Used:

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: PostgreSQL
* AI/ML: Python, Scikit-Learn
* Containerization: Docker
* Data Analysis: Jupyter Notebook

This project was developed as a Final Year Project to demonstrate the integration of artificial intelligence with modern full-stack web development for smart real estate investment solutions.
